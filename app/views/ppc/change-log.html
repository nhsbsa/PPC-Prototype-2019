Version 2

Page Change
/find-address “Find address” button removed, instead “Next” button performs find address function and links to list of addresses found.
/address-list
  • Changed from each address acting as a link to each address having a radio button and a “Next” button to submit choice.
  • “Address not listed” link added to send users to manual address entry. Rather than having to go back in browser and use the link on find address page.
  • Text formatting changed to align more with gov.
What is your address This page has been removed.
Instead users go straight from selecting address from list to NHS no question. Rather than first going to a page containing their chosen address pre-populated in input fields.
/duration and /payment-type Split into two separate question pages. Users are first asked to choose a duration of 3 or 12 months. If 12 months is chosen, the next page will ask if they wish to pay by direct debit or card payment. If 3 months chosen they skip to the next question as only card payment available.
/copy Checkboxes for email/mobile/no contact changed to two radio buttons. First radio button gives choice to be contacted by email and/or mobile and reveals input fields. Second radio button gives choice for no contact.
